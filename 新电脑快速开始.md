## ⚡ 第三步：测试工具

### 1. 打开工具（5秒）
```
双击：ai-ui-tool-v1.0-纯JS渲染版.html
浏览器自动打开
```

### 2. 配置API Key（10秒）
```
1. 点右上角 "⚙️ 设置"
2. 粘贴API Key: sk-or-v1-xxxxx
3. 点"保存"
4. 关闭设置
```

### 3. 快速测试（30秒）
```
1. 按F12打开Console（重要！）
2. 只勾选 ☑️ Gemini Image
3. 点"登录页"模板
4. 点"🚀 开始生成"
5. 观察Console输出
```

## 🎯 常见问题

### Q1: API Key在哪里？
```
A: OpenRouter网站
   https://openrouter.ai/
   登录 → Settings → API Keys
```

### Q2: Console怎么打开？
```
A: 按F12
   或右键 → 检查 → Console标签
```

### Q3: 如果所有模型都失败？
```
A: 检查顺序
   1. API Key正确吗？
   2. 网络正常吗？
   3. OpenRouter有余额吗？
   4. 只测Gemini（免费的）
```

### Q4: 文件太多记不住？
```
A: 只需要记住2个：
   1. HTML工具（用来测试）
   2. PROJECT-STATE（给AI看）
```
